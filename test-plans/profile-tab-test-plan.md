# Тест-план: раздел «Профиль» в Android-приложении

Чек-лист по мотивам реального опыта тестирования мобильного приложения. Названия экранов и логика адаптированы под гипотетическое приложение "AutoCare" (агрегатор услуг для автовладельцев), чтобы не раскрывать данные работодателя.

Платформа: Android
Фича: обновлённый экран «Профиль» — гостевой режим, авторизованный пользователь, настройки

## 1. Гостевой режим

- [ ] Отображается текст-заглушка "Вы в гостевом режиме. Войдите, чтобы разблокировать все функции"
- [ ] - [ ] Кнопка «Войти / Зарегистрироваться» — текст не обрезан, выравнивание корректно
- [ ] - [ ] Кнопка входа через Google — работает авторизация
- [ ] - [ ] Проверить отображение баннера с акциями в гостевом режиме
- [ ] - [ ] Точка входа в «Настройки» скрыта/недоступна для гостя
- [ ] - [ ] Проверить доступность приложения в гостевом режиме (нет краша, нет блокирующих экранов)

- [ ] ## 2. Шапка профиля — авторизованный пользователь

- [ ] - [ ] Аватар отображается (или заглушка в шахматную клетку, если фото не задано)
- [ ] - [ ] Полное имя и email отображаются корректно
- [ ] - [ ] Числовой ID пользователя отображается под блоком профиля (не редактируется)
- [ ] - [ ] Ссылка «Редактировать личные данные» кликабельна и открывает нужный экран

- [ ] ## 3. Раскладка и системная логика

- [ ] - [ ] Порядок виджетов сверху вниз соответствует макету (Гараж → Заказы → Промо-баннер → Избранное → Бонусы → Адрес доставки → Поддержка)
- [ ] - [ ] На экранах высотой 640px и ниже — элементы сжимаются пропорционально, заголовки и текст не перекрываются
- [ ] - [ ] При загрузке отображается skeleton loader на всех виджетах
- [ ] - [ ] При ошибке загрузки (таймаут >10с) — инлайн-сообщение "Не удалось загрузить [раздел]. Нажмите, чтобы повторить" — каждый виджет повторяет попытку независимо от других
- [ ] - [ ] Остальные виджеты остаются доступны, если один из них не загрузился
- [ ] - [ ] Страница скроллится плавно, нижняя навигация не перекрывает контент
- [ ] - [ ] Счётчики на бейджах виджетов соответствуют реальному количеству элементов
- [ ] - [ ] Отступы между блоками соответствуют макету
- [ ] - [ ] Ссылка на политику конфиденциальности кликабельна

- [ ] ## 4. Настройки

- [ ] - [ ] Поля «Имя» и «Email» заблокированы для редактирования на главном экране настроек
- [ ] - [ ] ID пользователя отображается как статичное поле
- [ ] - [ ] Присутствуют все пункты навигации: реквизиты, подписки, депозитный счёт, смена пароля, текущая страна, системные настройки, политика конфиденциальности, оценить приложение

- [ ] Подэкран «Личные данные»:
- [ ] - [ ] Поля «Имя» и «Фамилия» редактируемые
- [ ] - [ ] Поле Email — read-only
- [ ] - [ ] Кнопка «Сохранить» в тулбаре активируется только после изменения хотя бы одного поля
- [ ] - [ ] Нажатие на аватар открывает выбор действия: выбрать из галереи / сделать новое фото / удалить фото / отмена
- [ ] - [ ] Добавление фото из галереи работает
- [ ] - [ ] Съёмка фото с камеры работает
- [ ] - [ ] Загруженное фото обрезается в круг 56×56px без искажений
- [ ] - [ ] Загрузка файла не JPG/PNG или больше 5MB — выдаёт ошибку
- [ ] - [ ] «Удалить фото» — сбрасывает аватар на дефолтную заглушку
- [ ] - [ ] После возврата с экрана редактирования профиля — главный экран рендерится корректно

- [ ] Смена пароля:
- [ ] - [ ] Это отдельный экран, не встроенный в личные данные
- [ ] - [ ] Плейсхолдер поля пароля имеет корректные отступы
- [ ] - [ ] Блок смены пароля отображается корректно для гостевых пользователей

- [ ] Текущая страна:
- [ ] - [ ] Отображается с флагом
- [ ] - [ ] Нажатие открывает экран управления адресами

- [ ] ## 5. Оценить приложение

- [ ] - [ ] Выбор 4–5 звёзд → редирект в Google Play
- [ ] - [ ] Выбор 1–3 звёзд → открывает почтовый клиент с предзаполненным адресом поддержки
- [ ] - [ ] Без выбора оценки — блок остаётся в исходном состоянии

- [ ] ## 6. Виджет «Гараж»

- [ ] Счётчик бейджа:
- [ ] - [ ] Бейдж в шапке = количеству сохранённых автомобилей, скрыт при 0

- [ ] Пустое состояние:
- [ ] - [ ] Отображается форма добавления авто инлайн
- [ ] - [ ] Мотивационный заголовок "Добавьте автомобиль для точного подбора запчастей"

- [ ] Поиск по гос. номеру:
- [ ] - [ ] Флаг страны в поле поиска соответствует текущей стране пользователя
- [ ] - [ ] Кнопка поиска активна во время ввода
- [ ] - [ ] Успешный поиск → экран подтверждения с предзаполненной маркой/моделью
- [ ] - [ ] Неудачный поиск → инлайн "Автомобиль не найден. Введите данные вручную"

- [ ] Ручной ввод:
- [ ] - [ ] Три последовательных шага: марка → модель → модификация
- [ ] - [ ] Каждый следующий шаг заблокирован, пока не выбран предыдущий
- [ ] - [ ] Смена марки сбрасывает модель и модификацию
- [ ] - [ ] Кнопка «Добавить в гараж» активна только когда заполнены все три поля

- [ ] Обнаружение дублей:
- [ ] - [ ] Попытка добавить уже существующий автомобиль (тот же VIN или марка+модель+год+модификация) → инлайн "Этот автомобиль уже в вашем гараже", отправка блокируется

- [ ] Лимит карточек (1–20 автомобилей):
- [ ] - [ ] Карточки в один горизонтальный ряд с прокруткой
- [ ] - [ ] При ровно 20 автомобилях кнопка «+ Добавить машину» пропадает из шапки виджета
- [ ] - [ ] При попытке добавить 21-ю машину — тултип "Вы достигли максимального количества автомобилей"

- [ ] Карточка автомобиля:
- [ ] - [ ] Фото автомобиля (кастомное или по умолчанию)
- [ ] - [ ] Пробег с суффиксом "км"
- [ ] - [ ] Дата последнего ТО
- [ ] - [ ] Статус ТО: красный (просрочено), жёлтый (скоро), зелёный (всё ок)
- [ ] - [ ] Нажатие на карточку открывает Детали автомобиля
